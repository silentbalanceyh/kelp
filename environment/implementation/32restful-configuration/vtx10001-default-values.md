# Restful配置默认值

## 0. 对应关系

模型和数据表的对应关系如下

* PEAddress - EVX\_ADDRESS
* PERoute - EVX\_ROUTE
* PERule - EVX\_RULE
* PEUri - EVX\_URI
* PEVerticle - EVX\_VERTICLE
* PEScript - ENG\_SCRIPT
